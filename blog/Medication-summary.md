# 用药记录

这里是早上12点的我 现在时间是2月18号 距离昨天晚上想搞死自己已经过了... 6个小时

我来简单分析一下这用药的48小时:3次安非他酮 2次富马

在用药前 就是单纯的抑郁期加轻躁期

1. 抑郁期： 大脑处于**“低能量、低动力”**状态。那时候我虽然难受，但我的“执行功能”是瘫痪的。我连起床都觉得累，根本没有能量去构思一个复杂系统。那时的自杀倾向是“静息态”的，表现为想睡死过去、想不再醒来。

但是呢 现在轻躁期还在 但 抑郁期变成混合期了

2. 混合期： 药物（安非他酮）强行拉高了我的多巴胺和去甲肾上腺素。现在呢，拥有了**“躁狂的行动力”和“抑郁的绝望感”**。大概是这样的

    - 动力（Power）： 来自躁狂成分，能熬夜计算、研究公差、画草图。

    - 色调（Color）： 来自抑郁成分，目标依然指向死亡。

GAME OVER在这个阶段，从“愿望”变成了“工程任务”。 这是一个极其危险的组合啊：一个拥有高性能 CPU 的人，正在运行一个自毁程序。

最后呀 在混合期的高频思维下，大脑完成了一个**“绝望闭环”**：既然身体无法重塑，环境无法改变，学业即将崩溃，那么“物理毁灭”成了逻辑中最优、最严丝合缝的“解”。 最后 GAME OVER
